# Humanizer (Taiwan Traditional Chinese)

本專案為 [blader/humanizer](https://github.com/blader/humanizer) 的台灣繁體中文在地化改編版本。

這是一個用於移除 AI 生成寫作痕跡的 AI Agent 技能 (Skill)。在編輯或審閱文本時使用，使其讀起來更自然、更像人類所寫。
